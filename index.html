HTML

  • Header element contains the navigations of the webpage.
  • Head element contains the meta element; the information about the webpage.
  • Title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • Body element represents the visible content shown to the user.
  • Footer element are typically copyrights at the bottom of the page
  • Paragraph element represents a paragraph or block of text
  • Style element defines the CSS styles associated with a website.
  • "br" element creates a line of empty space, or a line break, between two blocks of content

CSS

  • CSS defines the colors, aesthetics, visual layout, and organization. We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options
  • Contains property: value; = color: blue;
  • Element selectors selects ALL elements that match the html tag (ie. h1, tag)
  • ID selectors should ONLY be used ONCE on the page. (#)
  • Class selectors selects all the elements that include the class name (.)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git stash: git stash temporarily shelves (or stashes) changes you've made to your working copy so you can work on something else, and then come back and re-apply them later on
  • git stash apply: unshelves changes
  • git add . : adds changes
  • git commit -m "": adds a commit message
  • git push origin branch: pushes branch to origin
  • git pull origin branch: pulls changes from specific branch
  • git log: shows logs
  • to add branch process: git pull, add, & push

JavaScript

  • A variable is a named container (i.e. a cup) that allows us to store data in our code
  • Variables are declared using special keywords (var, let, const.)
  • The data is the content inside the container (i.e.orange juice, water, soda)
  • Commom JS datatypes: string, Numeric, Boolean, Undefined, Object.
  • Data type String uses "", numeric data does not use quotation marks, boolean value only says true or false.
  • Control flow is the order in which the computer executes code in a script
  • conditional statements are used to make decisions based on conditions declared on the code.
  • Boolean: operators are used; ie. === equal value & equal type, != not equal
  • A function's name is followed by parentheses ()
  • console.log prints javascript in browser console
  • The "typeof" method prints the data types of a variable's value
